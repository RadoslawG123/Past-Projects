## 🔍 Detailed Project Descriptions

<details>
  <summary>📝 <b>Api-for-notes (Notes for goats)</b></summary>
  
  * **Tech:** Python, Flask, Flask-SQLAlchemy, Flask-Login, Bcrypt
  * **Description:** This is one of my most advanced mini-projects. It's a fully functional note-taking web application. Users can register and log in to their private accounts. Once authenticated, they can create, save, and read their personal notes (title + text). The sidebar provides quick access to previously saved notes.
  * **Key Takeaways:** Learned how to build a website in python, implement secure user authentication, password hashing (Bcrypt), and manage relational databases using SQLAlchemy ORM.
</details>

<details>
  <summary>🕷️ <b>Web-scraping</b></summary>

  * **Description:** A focused script built to automate data collection. It scrapes book data from `books.toscrape.com`, demonstrating my understanding of parsing HTML structures and automating web interactions in Python.
</details>

<details>

  * **Key Takeaways:** Successfully implemented Object-Oriented Programming (OOP) principles in a real, functional environment and improved my problem-solving skills by brainstorming game mechanics independently.
</details>
